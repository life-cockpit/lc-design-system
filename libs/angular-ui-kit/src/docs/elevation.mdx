## Usage

```css
box-shadow: var(--elevation-2);
border-radius: var(--borderRadius-md);
```

## Recommendations

| Element | Elevation | Border Radius |
|---|---|---|
| Cards | 1 | md (8px) |
| Dropdowns | 2 | md (8px) |
| Modals | 3 | lg (12px) |
| Tooltips | 4 | sm (4px) |
| Buttons | – | md (8px) |
| Inputs | – | md (8px) |
| Avatars | – | full |
| Badges | – | full |
