# Typography

The type system defines consistent font sizes, weights, and line heights.

## Font Families

| Token | Value |
|---|---|
| `fontFamily-base` | -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, ... sans-serif |
| `fontFamily-mono` | 'SF Mono', Monaco, Inconsolata, ... monospace |

## Line Heights

| Token | Value | Usage |
|---|---|---|
| `tight` | 1.25 | Headings, compact text |
| `normal` | 1.5 | Body text (default) |
| `relaxed` | 1.75 | Long-form reading text |

## Usage

```css
font-size: var(--typography-fontSize-base);
font-weight: var(--typography-fontWeight-semibold);
line-height: var(--typography-lineHeight-normal);
```

```html
<!-- Typography component -->
<lc-typography variant="h1">Heading</lc-typography>
<lc-typography variant="body">Body text</lc-typography>
<lc-typography variant="caption">Caption</lc-typography>
```
