# Spacing

The spacing system is based on a **4px grid** (0.25rem). Consistent spacing ensures visual harmony.

## Usage

```css
/* As CSS Custom Property */
padding: var(--spacing-4);       /* 16px */
margin-bottom: var(--spacing-6); /* 24px */
gap: var(--spacing-2);           /* 8px */
```

```html
<!-- With the Spacer component -->
<lc-spacer size="md"></lc-spacer>
```

## Recommendations

| Context | Recommended Spacing |
|---|---|
| Inside inputs | `spacing-2` (8px) – `spacing-3` (12px) |
| Between form fields | `spacing-4` (16px) – `spacing-5` (20px) |
| Between sections | `spacing-8` (32px) – `spacing-12` (48px) |
| Card padding | `spacing-4` (16px) – `spacing-6` (24px) |
| Icon to text | `spacing-2` (8px) |
| Button padding (horizontal) | `spacing-4` (16px) |
| Minimum touch target | `spacing-11` (44px) |
