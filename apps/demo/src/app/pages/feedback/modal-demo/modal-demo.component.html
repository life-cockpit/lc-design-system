Modal Component

Dialog overlays for focused user interactions requiring immediate attention.

Sizes

Modals come in five sizes: small, medium, large, extra-large, and full-screen.

Small Modal Medium Modal Large Modal Extra Large Modal Full Screen Modal 
<lc-modal [(open)]="showModal" size="lg" ariaLabel="Large modal">
  <!-- Modal content -->
</lc-modal>

<!-- Sizes: sm, md (default), lg, xl, full -->

Behavior Options

Control modal closing behavior with backdrop clicks and escape key.

No Backdrop Click Close No Escape Key Close No Close Button 
<lc-modal 
  [(open)]="showModal"
  [closeOnBackdrop]="false"
  [closeOnEscape]="false"
  [showCloseButton]="false"
  ariaLabel="Controlled modal"
>
  <!-- User must use provided actions to close -->
</lc-modal>

Form Example

Modals can contain forms and complex content.

Show Form Modal 
<lc-modal [(open)]="showModal" ariaLabel="Form modal">
  <div slot="header">
    <h2 class="text-xl font-semibold">Contact Form</h2>
  </div>
  <div slot="body">
    <form>
      <lc-input label="Name" [(ngModel)]="formData.name" />
      <lc-input label="Email" [(ngModel)]="formData.email" />
    </form>
  </div>
  <div slot="footer" class="flex gap-3">
    <lc-button variant="secondary" (clicked)="showModal.set(false)">
      Cancel
    </lc-button>
    <lc-button (clicked)="submitForm()">Submit</lc-button>
  </div>
</lc-modal>

Nested Modals

Modals can be nested, though this should be used sparingly.

Show Nested Modal 
<!-- Component Template -->
<lc-button (clicked)="showModal.set(true)">Open Modal</lc-button>

<lc-modal [(open)]="showModal" ariaLabel="My modal">
  <div slot="header">
    <h2 class="text-xl font-semibold">Modal Title</h2>
  </div>
  <div slot="body">
    <p>Modal content goes here.</p>
  </div>
  <div slot="footer">
    <lc-button (clicked)="showModal.set(false)">Close</lc-button>
  </div>
</lc-modal>

// Component Class
export class MyComponent {{ '{' }}
  showModal = signal(false);
{{ '}' }}
