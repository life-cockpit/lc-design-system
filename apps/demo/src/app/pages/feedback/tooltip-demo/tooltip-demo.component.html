Tooltip Directive

Contextual information displayed on hover or focus.

Basic Usage

Hover over or focus on elements to see tooltips.

Hover Me (Top) 
<lc-button lcTooltip="This is a tooltip">
  Hover Me
</lc-button>

Positions

Tooltips can be positioned in four directions.

Top Right Bottom Left 
<lc-button 
  lcTooltip="Tooltip on top" 
  tooltipPosition="top"
>
  Top
</lc-button>

<!-- Positions: top, right, bottom, left -->

With Delays

Control when tooltips appear and disappear.

Show Delay (500ms) Hide Delay (300ms) Both Delays 
<lc-button 
  lcTooltip="Shows after 500ms" 
  [tooltipShowDelay]="500"
  [tooltipHideDelay]="300"
>
  Delayed Tooltip
</lc-button>

Disabled State

Tooltips can be disabled conditionally.

Disabled Tooltip Enabled Tooltip 
<lc-button 
  lcTooltip="This tooltip can be disabled" 
  [tooltipDisabled]="isDisabled"
>
  Conditional Tooltip
</lc-button>

On Different Elements

Tooltips work on various HTML elements.

Button Link Badge 
<!-- Button -->
<lc-button lcTooltip="Button tooltip">Button</lc-button>

<!-- Link -->
<a href="#" lcTooltip="Link tooltip">Link</a>

<!-- Input -->
<input 
  type="text" 
  lcTooltip="Input tooltip"
  placeholder="Hover or focus me"
/>

<!-- Any element -->
<span lcTooltip="Span tooltip">Hover text</span>

Long Text

Tooltips can contain longer text and will wrap appropriately.

Long Tooltip 
<lc-button lcTooltip="This is a much longer tooltip...">
  Long Tooltip
</lc-button>
