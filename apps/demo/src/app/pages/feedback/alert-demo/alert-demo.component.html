Alert Component

Inline notifications to display important messages with semantic variants.

Variants

Success! Your changes have been saved successfully. Error! Something went wrong. Please try again. Warning! This action cannot be undone. Info: New features are available in this release. 
<lc-alert variant="success">
  <strong>Success!</strong> Your changes have been saved.
</lc-alert>

<lc-alert variant="error">
  <strong>Error!</strong> Something went wrong.
</lc-alert>

<lc-alert variant="warning">
  <strong>Warning!</strong> This action cannot be undone.
</lc-alert>

<lc-alert variant="info">
  <strong>Info:</strong> New features are available.
</lc-alert>
