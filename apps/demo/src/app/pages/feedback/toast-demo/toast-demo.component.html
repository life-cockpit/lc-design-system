Toast Component

Ephemeral notifications that appear temporarily to provide feedback.

Setup

Import the ToastService and inject it into your component. 

import {{ '{' }} ToastService {{ '}' }} from '@lc-platform/shared/ui-kit';

export class MyComponent {{ '{' }}
  constructor(private toastService: ToastService) {{ '{' }}{{ '}' }}
{{ '}' }}

Variants

Show Success Toast Show Error Toast Show Warning Toast Show Info Toast 
this.toastService.show({{ '{' }}
  message: 'Your changes have been saved',
  variant: 'success',
  duration: 3000
{{ '}' }});

With Action

Toasts can include an action button for user interaction.

Show Toast with Action 
this.toastService.show({{ '{' }}
  message: 'Connection lost',
  variant: 'error',
  action: {{ '{' }}
    label: 'Retry',
    handler: () => this.reconnect()
  {{ '}' }}
{{ '}' }});

Positions

Toasts can be positioned in six different locations on the screen.

Top Left Top Center Top Right Bottom Left Bottom Center Bottom Right 
this.toastService.show({{ '{' }}
  message: 'Notification',
  variant: 'info',
  position: 'top-right' // or: top-left, top-center, bottom-left, bottom-center, bottom-right
{{ '}' }});

Duration

Control how long toasts remain visible. Set duration to 0 for persistent toasts.

Show Persistent Toast (Manual Close) 
this.toastService.show({{ '{' }}
  message: 'This toast stays until manually closed',
  variant: 'info',
  duration: 0 // 0 = persistent, must be manually dismissed
{{ '}' }});

Multiple Toasts

The toast service manages a queue and limits the number of visible toasts.

Show Multiple Toasts Clear All Toasts 
// Clear all visible toasts
this.toastService.clearAll();
