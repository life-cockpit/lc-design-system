Responsive Grids

Grids that adapt to screen size using breakpoint prefixes.

1 Column → 2 Columns → 3 Columns

@for (i of [1, 2, 3, 4, 5, 6]; track i) {
Item {{ i }}
} 
<div class="grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4">

1 Column → 3 Columns → 4 Columns

@for (i of [1, 2, 3, 4, 5, 6, 7, 8]; track i) {
Item {{ i }}
} 
<div class="grid grid-cols-1 md:grid-cols-3 lg:grid-cols-4 gap-4">

Auto-Fit Grids

Automatically fill available space with items.

}

Best Practices

  • ✓ Use Tailwind's grid utilities instead of creating custom grid components
  • ✓ Start with mobile-first layouts (grid-cols-1) and add responsive breakpoints
  • ✓ Use col-span-* for items that need to span multiple columns
  • ✓ Consider auto-fill/auto-fit for dynamic content that should flow naturally
  • ✓ Use gap-* utilities for consistent spacing between items
