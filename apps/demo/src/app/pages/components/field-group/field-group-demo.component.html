Field Group

Field Group displays label-value pairs in a consistent, readable format. Ideal for profile pages, detail views, and read-only data displays.

Interactive Playground

@if (showIcon()) {
}

Basic Usage

Simple label-value display without icons.

 
<lc-field-group label="Email" value="user@example.com"></lc-field-group>

With Icons

Add visual context with icons from Tabler Icons.

 
<lc-field-group
  label="Language"
  value="English (US)"
  icon="globe-alt">
</lc-field-group>

Compact Mode

Reduced padding for denser layouts.

 
<lc-field-group
  label="Status"
  value="Active"
  icon="check-circle"
  [compact]="true">
</lc-field-group>

API Reference

Inputs

Property Type Default Description
label string required Label text displayed above the value
value string — Value text to display (optional if using content projection)
icon string — Optional icon name from Tabler Icons
iconSize 'xs' | 'sm' | 'md' 'sm' Size of the icon
compact boolean false Use compact styling with less padding
