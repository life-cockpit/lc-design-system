Typography Component

The Typography component provides consistent text styling with semantic HTML elements (h1-h6, p, span), multiple variants, color options, alignment, font weights, text transforms, and line clamping. Based on the Life-Cockpit design system using Open Sans font family.

Heading Variants

Typography variants h1-h6 render semantic HTML heading elements with sizes based on lc-design.md specification.

Font Weights

Open Sans supports multiple weights: Regular (400), Medium (500), Semibold (600), and Bold (700).

Regular weight (400) - The default weight for body text Medium weight (500) - Slightly bolder than regular Semibold weight (600) - Used for headings and emphasis Bold weight (700) - Strongest emphasis and large headings

Text Transform

Transform text case with uppercase, lowercase, or capitalize options.

No transform (default) - Text appears as written Uppercase transform - ALL LETTERS ARE CAPITALIZED Lowercase transform - all letters are lowercase Capitalize transform - First Letter Of Each Word Is Capitalized 
<lc-typography transform="uppercase">Text</lc-typography>
<lc-typography transform="capitalize">Text</lc-typography>
