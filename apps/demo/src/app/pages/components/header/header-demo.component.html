Header Component

AppHeaderComponent provides persistent top navigation with flexible branding (logo or title/subtitle), user profile dropdown with optional theme toggle, and hamburger menu for mobile sidebar. Unified component for all application headers.

Inputs

Outputs

Usage

import { HeaderComponent } from '@life-cockpit/angular-ui-kit';

<lc-header
  [logo]="'/assets/logo.svg'"
  [title]="'Life-Cockpit'"
  [subtitle]="'User Profile'"
  [userName]="'John Doe'"
  [userEmail]="'user@example.com'"
  [showHamburger]="true"
  [showThemeButton]="true"
  [showProfileMenuItem]="true"
  (hamburgerClick)="toggleSidebar()"
  (themeToggleClick)="toggleTheme()"
  (profileClick)="navigateToProfile()"
  (logoutClick)="handleLogout()"
/>

Features

  • ✓ Clickable logo for home navigation
  • ✓ Optional title and subtitle display
  • ✓ User profile dropdown with email, Profile link, and Logout
  • ✓ Optional theme toggle in menu
  • ✓ Hamburger menu toggle for mobile sidebar
  • ✓ OnPush change detection for performance
  • ✓ Design system tokens for styling
  • ✓ Keyboard accessible (Tab navigation, Enter/Space activation)
  • ✓ ARIA attributes for screen readers
