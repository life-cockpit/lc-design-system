Sidenav Component

A drawer navigation component that slides in from the left or right side of the screen. Includes overlay backdrop, keyboard navigation (Escape to close), and accessibility features.

Default Left Sidenav

A basic sidenav that slides in from the left side with an overlay backdrop. Click the overlay or press Escape to close.

With Navigation Items

Sidenav can display navigation items using NavigationItem[] interface with icons, labels, routes, and active state highlighting.

Open Navigation Sidenav
Current active route: {{ activeRoute() }} 
navItems: NavigationItem[] = [
  {
    id: 'home',
    icon: 'lc-icon-home',
    label: 'Home',
    route: '/',
    displayOrder: 1,
  },
  {
    id: 'trading',
    icon: 'lc-icon-chart',
    label: 'Trading',
    route: '/trading',
    requiredRole: 'LC.Trader',
    displayOrder: 2,
  },
];

<lc-sidenav
  [itemsInput]="navItems"
  [activeRouteInput]="activeRoute"
  (itemClicked)="handleNavItemClick($event)">
</lc-sidenav>

Accessibility Features

  • ARIA role="navigation" for semantic meaning
  • ARIA aria-modal="true" to indicate modal behavior
  • ARIA aria-label for screen reader description
  • Keyboard navigation: Press Escape to close
  • Focus management with close button clearly accessible
  • Click overlay backdrop to close
  • Respects prefers-reduced-motion for animations
