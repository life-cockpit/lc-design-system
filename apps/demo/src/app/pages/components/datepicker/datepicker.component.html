DatePicker Component

Date selection component with calendar overlay and keyboard navigation.

Basic DatePicker

The default datepicker with outline variant.

<lc-datepicker
  label="Select Date"
  placeholder="Choose a date"
  helperText="Pick any date from the calendar"
  [(ngModel)]="date"
/>

Variants

DatePicker supports outline and filled variants.

Outline (default)

Filled

<lc-datepicker [variant]="'outline'" />
<lc-datepicker [variant]="'filled'" />

Sizes

Available sizes: xs, sm, md, lg

Extra Small (xs)

Small (sm)

Medium (md)

Large (lg)

<lc-datepicker [size]="'xs'" />
<lc-datepicker [size]="'sm'" />
<lc-datepicker [size]="'md'" />
<lc-datepicker [size]="'lg'" />

States

DatePicker supports various interactive states.

Disabled

Error

Readonly

Required

<lc-datepicker [disabled]="true" />
<lc-datepicker [error]="true" errorMessage="Error message" />
<lc-datepicker [readonly]="true" />
<lc-datepicker [required]="true" />

Date Constraints

Restrict selectable dates with min and max date boundaries.

Min/Max Dates

<lc-datepicker
  [minDate]="minDate"
  [maxDate]="maxDate"
/>

Date Format

Customize the display format of selected dates.

Custom Format

<lc-datepicker [format]="'MM/dd/yyyy'" />
<lc-datepicker [format]="'dd.MM.yyyy'" />
<lc-datepicker [format]="'yyyy-MM-dd'" />

Reactive Forms Integration

DatePicker works seamlessly with Angular reactive forms.

<form [formGroup]="form">
  <lc-datepicker formControlName="startDate" />
  <lc-datepicker formControlName="endDate" />
  <lc-datepicker formControlName="birthDate" />
</form>
