Menu Component

Dropdown menus for navigation, actions, and user profiles. Built with content projection and keyboard navigation.

Basic Menu

A simple dropdown menu with text items and a danger variant.

Open Menu 
<lc-menu
  [items]="menuItems"
  [isOpen]="menuOpen"
  (itemClick)="handleItemClick($event)"
  (closed)="menuOpen = false"
>
  <lc-button trigger (clicked)="menuOpen = !menuOpen">
    Open Menu
  </lc-button>
</lc-menu>

// Component
menuItems: MenuItem[] = [
  {{ '{' }} id: 'item1', label: 'Menu Item 1' {{ '}' }},
  {{ '{' }} id: 'item2', label: 'Menu Item 2' {{ '}' }},
  {{ '{' }} id: 'item3', label: 'Menu Item 3', dividerAfter: true {{ '}' }},
  {{ '{' }} id: 'delete', label: 'Delete', variant: 'danger' {{ '}' }},
];

Profile Menu with Metadata

Use metadata to display additional information like subtitles or badges. Perfect for user profile menus. 

menuItems: MenuItem[] = [
  {{ '{' }}
    id: 'profile',
    label: 'John Doe',
    icon: 'user',
    metadata: {{ '{' }} subtitle: 'john.doe@example.com' {{ '}' }},
    dividerAfter: true
  {{ '}' }},
  {{ '{' }} id: 'account', label: 'Account Settings', icon: 'cog-6-tooth' {{ '}' }},
  {{ '{' }} id: 'billing', label: 'Billing', icon: 'credit-card' {{ '}' }},
  {{ '{' }} id: 'logout', label: 'Logout', icon: 'arrow-right-start-on-rectangle', variant: 'danger' {{ '}' }},
];

Position Variants

Control menu position relative to the trigger: bottom-left, bottom-right, top-left, or top-right.

Bottom Left Bottom Right Top Left Top Right 
<lc-menu position="bottom-left">...</lc-menu>
<lc-menu position="bottom-right">...</lc-menu>
<lc-menu position="top-left">...</lc-menu>
<lc-menu position="top-right">...</lc-menu>

MenuItem Interface

interface MenuItem {{ '{' }}
  id: string;                    // Unique identifier
  label: string;                 // Display text
  icon?: string;                 // Heroicon name
  iconSize?: 'xs' | 'sm' | ...  // Icon size
  href?: string;                 // Link URL
  target?: '_blank' | ...        // Link target
  variant?: 'default' | 'danger'; // Style variant
  dividerAfter?: boolean;        // Show divider after item
  disabled?: boolean;            // Disable item
  metadata?: {{ '{' }}
    subtitle?: string;           // Secondary text
    badge?: string;              // Badge text
  {{ '}' }};
{{ '}' }}

Features

  • ✓ Content projection slots for trigger, header, and footer
  • ✓ Support for both link (href) and button menu items
  • ✓ Icons with configurable size
  • ✓ Danger variant for destructive actions
  • ✓ Dividers between menu sections
  • ✓ Metadata support (subtitle, badge)
  • ✓ Click outside to close
  • ✓ Escape key to close
  • ✓ Smooth slide-down animation
  • ✓ Dark mode support
