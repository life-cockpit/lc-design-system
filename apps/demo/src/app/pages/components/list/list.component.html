Best Practices

  • Use vertical layout for navigation menus and content lists
  • Use horizontal layout for tab-like navigation or toolbars
  • Enable dividers for better visual separation in long lists
  • Use icons to provide visual context for list items
  • Keep action buttons concise (1-2 words)
  • Provide unique IDs for items when using click events
  • Use disabled state for items that are temporarily unavailable
  • Limit horizontal lists to 3-5 items for better UX
