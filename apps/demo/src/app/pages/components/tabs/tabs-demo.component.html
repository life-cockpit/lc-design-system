Keyboard Navigation

Tabs support full keyboard navigation for accessibility.

Keyboard Shortcuts

  • Tab Focus the tab list
  • Arrow Left/Right Navigate between tabs (horizontal)
  • Arrow Up/Down Navigate between tabs (vertical)
  • Home Jump to first tab
  • End Jump to last tab

      <lc-tab label="Tab 2">Content 2</lc-tab>
      <lc-tab label="Tab 3">Content 3</lc-tab>

Programmatic Selection

Control tab selection programmatically using the selectedIndexInput property.

Step 1: Setup

Configure your initial settings.

Step 2: Configuration

Customize your preferences.

Step 3: Complete

This tab is selected by default (index 2).

 
// TypeScript
@Component({
  selector: 'app-wizard',
  template: \`
    <lc-tabs [selectedIndexInput]="currentStep">
      <lc-tab label="Step 1">Step 1 content...</lc-tab>
      <lc-tab label="Step 2">Step 2 content...</lc-tab>
      <lc-tab label="Step 3">Step 3 content...</lc-tab>
    </lc-tabs>
  \`
})
export class WizardComponent {
  currentStep = 2; // Start at step 3 (index 2)
}
