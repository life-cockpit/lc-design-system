Pagination Component

Navigate through pages of content with customizable page sizes and size variants

Default Pagination

Basic pagination with 250 total items, 10 items per page, showing 25 pages 

<lc-pagination
  [currentPageInput]="currentPage"
  [totalItemsInput]="250"
  [pageSizeInput]="10"
  (pageChange)="onPageChange($event)"
></lc-pagination>

Size Variants

Three size options: sm, md (default), lg

Small

<lc-pagination [sizeInput]="'sm'" ... ></lc-pagination>

Medium (Default)

<lc-pagination [sizeInput]="'md'" ... ></lc-pagination>

Large

<lc-pagination [sizeInput]="'lg'" ... ></lc-pagination>

Dynamic Page Sizes

Allow users to select how many items per page. Current: {{ currentPage5() }} of {{ totalPages5 }} 

<select (change)="onPageSizeChange($event)">
  <option value="10">10</option>
  <option value="25">25</option>
  <option value="50">50</option>
  <option value="100">100</option>
</select>

<lc-pagination
  [currentPageInput]="currentPage"
  [totalItemsInput]="500"
  [pageSizeInput]="pageSize()"
  (pageChange)="onPageChange($event)"
></lc-pagination>

Many Pages with Ellipsis

When there are many pages (100 in this example), ellipsis (...) are used to truncate the page list 

<lc-pagination
  [currentPageInput]="42"
  [totalItemsInput]="1000"
  [pageSizeInput]="10"
  [maxVisiblePagesInput]="7"
  (pageChange)="onPageChange($event)"
></lc-pagination>

With Item Count Info

Display the range of items being shown and total count 

<lc-pagination
  [currentPageInput]="currentPage"
  [totalItemsInput]="235"
  [pageSizeInput]="20"
  [showInfoInput]="true"
  (pageChange)="onPageChange($event)"
></lc-pagination>

Visual Focus Indicators

All interactive elements have clear focus indicators that meet WCAG 2.1 AA contrast requirements
