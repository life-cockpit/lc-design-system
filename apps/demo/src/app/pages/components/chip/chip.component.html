Best Practices

  • Use chips for displaying removable selections, filters, or tags
  • Keep chip labels concise (1-3 words)
  • Use icons sparingly to add meaning or visual distinction
  • Choose variant colors that match the semantic meaning (e.g., error for validation issues)
  • For fixed tags that shouldn't be removed, set removable="false"
  • Provide visual feedback when chips are removed (e.g., animations or messages)
