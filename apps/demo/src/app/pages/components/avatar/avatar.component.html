Best Practices

  • Always provide a name for accessibility and initials fallback
  • Use appropriate size for the context (xs/sm for lists, lg/xl for profiles)
  • Include alt text when using images for better accessibility
  • Use status indicators sparingly to show real-time user presence
  • Ensure images are optimized and appropriate size (200x200px recommended)
  • Handle image load errors gracefully - component auto-falls back to initials
