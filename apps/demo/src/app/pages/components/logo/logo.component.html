Logo Component

Display the Life-Cockpit logo or emblem with customizable size. Supports both full logo with text and emblem-only variants.

Variants

The logo component supports two variants: full logo with text and emblem-only.

Full Logo
Emblem 
<!-- Full Logo with text -->
<lc-logo variant="full" size="md" />

<!-- Emblem only -->
<lc-logo variant="emblem" size="md" />

Sizes

Logos come in five sizes: xs (24px), sm (32px), md (48px), lg (64px), and xl (96px).

Clickable State

Add the clickable property to make the logo appear interactive with hover effects.

Default
Clickable (Hover me) 
<!-- Clickable logo with hover effect -->
<lc-logo variant="full" size="md" [clickable]="true" />

Common Use Cases

Application Header

<header>
  <lc-logo variant="full" size="md" [clickable]="true" />
</header>

Compact Navigation

<nav class="compact-nav">
  <lc-logo variant="emblem" size="sm" [clickable]="true" />
</nav>

Authentication Pages

<div class="auth-card">
  <lc-logo variant="full" size="lg" class="auth-logo" />
  <form>...</form>
</div>

Accessibility

The logo component includes accessibility features:

  • Alt text: Customize the alt attribute for screen readers (default: "Life-Cockpit")
  • Proper semantics: Uses standard <img> tag for logo display
  • Clickable indication: When clickable is true, cursor changes to pointer
<!-- Custom alt text for better context -->
<lc-logo variant="full" size="md" alt="Return to homepage" />
