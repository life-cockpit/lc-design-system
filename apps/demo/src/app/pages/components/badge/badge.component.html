Best Practices

  • Use rounded badges for counts: Pill-shaped badges work best for displaying numeric values
  • Choose appropriate sizes: Match badge size to surrounding content (e.g., small badges for inline text)
  • Limit text length: Badges should be compact - use abbreviations if necessary
  • Use semantic variants: Apply success/warning/error variants to convey status meaningfully
  • Avoid overuse: Too many badges can create visual clutter
