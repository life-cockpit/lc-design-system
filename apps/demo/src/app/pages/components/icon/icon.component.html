Icon Component

Display Heroicons with customizable size, variant, and color. Supports both outline and solid styles with full accessibility.

Sizes

Icons come in five sizes: xs (16px), sm (20px), md (24px), lg (32px), and xl (40px).

@for (size of sizes; track size) {
{{ size }}
} 
<!-- Extra Small (16px) -->
<lc-icon name="star" size="xs" />

<!-- Small (20px) -->
<lc-icon name="star" size="sm" />

<!-- Medium (24px) - Default -->
<lc-icon name="star" size="md" />

<!-- Large (32px) -->
<lc-icon name="star" size="lg" />

<!-- Extra Large (40px) -->
<lc-icon name="star" size="xl" />

Variants

Icons support outline (default) and solid variants from Heroicons.

Outline
Solid 
<!-- Outline (default) -->
<lc-icon name="heart" variant="outline" />

<!-- Solid -->
<lc-icon name="heart" variant="solid" />

Colors

Customize icon colors using hex values, CSS variables, or Tailwind classes.

Current
Primary
Success
Error
Warning 
<!-- Current color (default) -->
<lc-icon name="check-circle" color="currentColor" />

<!-- Hex color -->
<lc-icon name="check-circle" color="#208497" />

<!-- CSS variable -->
<lc-icon name="check-circle" color="var(--color-primary-500)" />

<!-- Tailwind class -->
<lc-icon name="check-circle" color="text-primary-500" />
