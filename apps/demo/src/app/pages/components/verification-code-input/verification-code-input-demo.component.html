API Reference

Features

  • Auto-Focus: Automatically moves to the next input after entering a digit
  • Backspace Navigation: Pressing backspace on an empty input moves to the previous one
  • Paste Support: Detects pasted 6-digit codes and automatically fills all inputs
  • Auto-Submit: Optional automatic submission when all digits are entered
  • Numeric Only: Automatically filters out non-numeric characters
  • Reactive Forms Integration: Full ControlValueAccessor implementation
  • Accessibility: Proper ARIA labels and keyboard navigation
  • Error States: Visual error feedback with error messages

Common Use Cases

Email Verification

Verify user email addresses during registration or login by sending a 6-digit code.

Two-Factor Authentication

Add an extra layer of security with time-based verification codes.

Password Reset

Verify identity before allowing users to reset their passwords.

Phone Verification

Confirm phone numbers by sending verification codes via SMS.
